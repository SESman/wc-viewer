Help and Instructions

This application was developed to provide scientists a simple interface for exploring telemetry data from tags deployed on marine wildlife. At this time, only a subset of tag/data types from Wildlife Computers is supported.

Getting Started

Wildlife Computers Data Portal

All data uploaded to the application must have been processed on the Wildlife Computers Data Portal. For best results, any deployments loaded should have been properly setup with correct start/end times and any settings files uploaded to the portal.

There are two methods for providing data:

WC Data Portal Zip File

Upload a *.zip file downloaded from the Wildlife Computers Data Portal. Only one deployment/file can be explored at a time.The name of the *.zip file should include only the PTT integer (e.g. 12345.zip).

Get Data via Web Services

If you have enabled web services on your Wildlife Computers Data Portal account, you can provide a PTT identifier and a web key file. The application will pull data directly from the portal.

GPS/Fastloc Tags

Fastloc tags with GPS location data are supported. However, the automatic location processing should be turned on in the Wildlife Computers Data Portal for the deployment. This will create a special Locations.csv file that includes both Argos locations and GPS locations determined from the default Fastloc process.

Exploring the Data

The application will process and display both location and any behavior data recieved.

Interacting with the Map

The map will initially display all locations. The map can be zoomed by either using the '+' and '-' icons or the mouse wheel (or equivalent trackpad action). The icons will zoom the map based on the current center point. The mouse wheel will zoom dependent on the current cursor location. Double clicking the map map will zoom in one step at that location. Zooming the map has no filtering impact on the display of other data (e.g. zooming into a subset of locations does not subset the dive behavior data shown).

Interacting with Behavior Data

All of the behavior and other data from the deployment are displayed as interactive timeseries graphs. Dragging the mouse will create a brush to zoom in on a region of time. The time series of Argos messages has an additional range selector that can be used to refine the date window of interest. All of the time series data and the map data are synced. Changes in any of the time series windows will sync with all the other windows and also subset the map data. Double click on any of the behavior data graphs to return to the full view.
